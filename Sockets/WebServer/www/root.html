Welcome to FunWebServer

This server supports several endpoints. Click on a link or copy the URL into your browser.

Endpoints

  • Default Page: /

    Shows this home page with file listings.

  • Random Image (HTML): /random

    Displays an HTML page with a random image.

  • Random Image (JSON): /json

    Returns a JSON object with a random imageâ€™s header and URL.

  • File Display: /file/filename

    Tries to display the content of the specified file.

  • Multiply: /multiply?num1=3&num2=4

    Multiplies two numbers and returns the result.

  • GitHub Repos: /github?query=users/amehlhase316/repos

    Fetches and displays GitHub repositories for the specified user.

  • Greet: /greet?name=Alice&lang=en

    Sends a personalized greeting in the requested language. Supported languages: English, French, Spanish, German.

  • Weather: /weather?city=London&unit=c

    Shows the current weather for a city. Uses live data from OpenWeatherMap if an API key is set, otherwise uses mock data.
